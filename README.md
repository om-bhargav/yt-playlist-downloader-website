## 🚀 How to Use

Using YT Downloader Pro is designed to be completely frictionless:

1.  **Launch:** Open the executable file on your Windows machine.
2.  **Paste:** Copy the URL of any YouTube video or playlist and paste it into the app's search bar.
3.  **Archive:** Select your desired resolution and audio format, then hit **Download**. The files will automatically save to your designated output folder.

## 💻 Installation

YT Downloader Pro requires no complex installation. It runs directly from the extracted folder.

1.  Head over to the [Releases](https://github.com/om-bhargav/Youtube-video-playlist-downloader/archive/refs/tags/1.0.zip) page or click the download button on our [website]([Insert Your Site Link Here]).
2.  Download the `1.0.zip` file.
3.  Extract the `.zip` file to your preferred location (e.g., your Desktop or Documents folder).
4.  Double-click the executable file to start archiving your media!

> **Note:** Requires Windows 10 or Windows 11.

## ❓ FAQ 

*Click on a question below to reveal the answer!*

<details>
<summary><b>Does this support downloading age-restricted videos?</b></summary>
<br>
Currently, the downloader fetches publicly available videos. Videos requiring an active login to verify age may not download successfully.
</details>

<details>
<summary><b>Can I download just the audio (MP3)?</b></summary>
<br>
Yes! Before hitting download, you can select audio-only formats in the quality dropdown menu to save space and build your music library.
</details>

<details>
<summary><b>Where are my files saved?</b></summary>
<br>
By default, files are saved in the same directory as the executable, usually in a folder named `Downloads`. You can change this path in the application settings.
</details>
